# Ovo Recurring Payment

## Clone

> git clone https://github.com/muntasirrahman/doku-ovo-proto3.git

## Install

To install dependencies, type here

> npm install

## Run

> npm run start

Result:

    {
      "code": "invalid_signature",
      "message": "Invalid Header Signature",
      "type": "invalid_request_error"
    }
